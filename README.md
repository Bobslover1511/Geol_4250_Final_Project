# Geol_4250_Final_Project
Repository for Geol4250 final project. Forward modelling and Inversion of EM data collected for mapping of taliks over continuous permafrost were done. Cole-Cole model was implemented to look for any IP effects. This repository contains 3 notebooks: 2301 Forward modelling; 2301 Inversion; 2301 iP.
## Download Anaconda
  https://www.anaconda.com/download
## Make conda environment
  on command line type:
  
  conda create --name geol_4250_final_project python=3.9 numpy scipy pandas geopandas jupyter git

  activate SimPEG:

  conda install --channel conda-forge simpeg
  
## Open jupyter lab
  on command line:
  
  conda activate geol_4330
  
  jupyter lab
